Mikicrep Studios - Neofetch

Available on linux

Latest release: 1.1

This is fork of neofetch, but with most of its code removed so its fast and only logo is Mikicrep OS.
Neofetch is CLI tool which allows you to get information about your PC.
Download is only possible by cloning our github repo, please do not use release from Releases page.

Features:

  • Easy accessing of PC specs
  • Written in bash
  • Very very lightweight
