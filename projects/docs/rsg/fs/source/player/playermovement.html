/source/player/playermovement.cpp - Explanation

This file is used to handle pressing WASD to move player

Refrenced in player.h

player::Movement()

  • This function handles WASD keys

    • W moves up

    • A moves left

    • S moves down

    • D moves right

  1. Detects if W/A/S/D key pressed

  2. Checks if target block is solid

  3. Checks if player will go off bounds of map

  4. Check if player is standing on color 0 (air), if yes then set air on that block

  5. If not then player is standing on solid block (because it is impossible to get block with color 0 and destroying block sets color back to 0) and then places block

  6. Player changes position
